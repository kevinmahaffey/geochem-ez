Calculating pH-

For more help: Load the file called "Example 3 Calculate pH.geo" to see how to run these types of cases.

This is an option that is accessed from the Advanced Options area. Most users will be making solutions with a known pH and will not opt for having Geochem make this calculation. However, if the user cares to use this function, then take the following steps after opening the Advanced Options area:

1) Enter a Guessed pH- when Geochem will calculate the pH, the system needs a guess. Enter an approximate pH value.
