Metal / Ligand Database-

Geodat-b.lig is the default database. Advanced users can modify the database, modifying the constants to suit a particular system, and adding more ligands to the database. Note that there are some open slots for ligands.

The program utilizes this database to calculate the interactions between the various ligands and metals in the solution being considered. The database is divided into four main sections. The first part is a list of metals and ligands along with their numerical program codes. The second part is a list of the reaction constants (log K values) between particular metal-ligand pairs, allowing for simple solids (up to 3) and soluble complexes (up to 6) that would occur due their interaction. The third section is for redox reactions, containing constants (log K values) for 29 different redox reactions. The fourth part contains information for calculating mixed solids.
