Infinite-Dilution Output Table-

The table of infinite-dilution (activity-based) constants lists all of the thermodynamic stability constants used, in a format similar to that of the metal / ligand database.Â Each row begins with two pairs of numbers (4 3, 4 99, 1 3, etc.) which represent the metal (M) and ligand (L) pair, followed by log K values for simple solids (up to 3) and soluble complexes (up to 6).

In the case shown below, the first line of display is: M L Solid 1 Solid 2 Solid 3 Complex 1 Complex 2 Complex 3 Complex 4 Complex 5

Here, 4 3 is the interaction of K+ with Cl-, with no log K values for simple solids, and one log K value for a soluble complex. See metal / ligand database for the list of metal and ligand code numbers.

The Infinite Dilution output table is generated automatically during the calculations and can not be deselected through Advanced Options, however it can be removed from the Geochem-EZ output file by excluding it in the "Output Viewer". Please see Output Tables for more information on this topic.
