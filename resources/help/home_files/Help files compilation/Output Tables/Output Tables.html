Output Tables-

See also: Advanced Options and Run Case for more information on this topic.

By default, Geochem-EZ displays three output tables-Case Progress, Primary Percentage Distribution, and Overall Speciation. The user can opt out of displaying any of these tables by opening the "Advanced Options" area ("Tables to be generated for this run") and unchecking the box beside that output table. Conversely, one can add other outputs (Conditional Constants, Soluble Complexes) to the case results by checking off their boxes in the "Advanced Options" area. All of the chosen tables will be generated when the case is run. However, one does not necessarily have to include any of these output tables when an Geochem-EZ output file is created and saved. Simply choose / highlight the name of the output table in the "Output Viewer " and click on the "Exclude from output" button in the lower right side corner. Repeat this step until only the desired tables have been left. After the user has chosen the output tables to include in Output file, then click on "Save Output File". This will open the " SAVE" dialog box for naming and locating the Output file.

An Infinite Dilution table is automatically generated as part of the Geochem program, regardless of what output tables you have chosen in the "Advanced Options" area ("Tables to be generated for this run"). However, it can be excluded from the output file by highlighting its name in the "Output Viewer " and clicking on the "Exclude from output" button in the lower right side corner of the dialog box.
