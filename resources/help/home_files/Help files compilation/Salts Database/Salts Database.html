The "Edit" menu bar allows the user to build and edit the salts database. Tailor the database to your needs, adding salts that are used frequently and removing those that are not.

Salts Database- a file (Geodat-A.salts) which contains information on salts commonly used to make nutrient solutions. The database is constructed so that each row describes a particular salt and parses it into its constituent metal and ligand parts. Included in this descriptor row are the molecular weight of the salt, the metal and ligand codes (see "metal / ligand database"), and the stoichiometries of the metals and ligands in that salt. Salts may be added to the database using the "Create a New Salt" function.

Create A New Salt- allows the user to add to the Salts database, as long as the metals and ligands that make up the salt are in the metal / ligand database. Click on "Create a New Salt" and the "Salt Creator" dialog box opens. The user can name the salt and then enter the formula manually, using the "Generate" button to parse the compound. Alternatively, using the drop down menu one can select the metals and ligands and enter the amount of the particular ion. For example, in creating the salt Na2HPO4 one could use the drop down menu to select Na+ and enter 2 as the amount, since for every mole of Na2HPO4 there are two mole equivalents of Na+. After that H+ would be entered on the next line, with one as the amount. The same goes for PO4-. When all of the components of the salt have been added, then click on "Save" at the bottom of the "Salt Creator" dialog box and this formulation will be added to the Salts database.

Edit Salts Database- use this feature to remove or to modify the information about a salt presently in the Salts database.
