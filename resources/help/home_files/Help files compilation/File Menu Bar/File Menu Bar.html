File Menu Bar-

New Run- adds another interface. This may useful to users who which to create cases with components different than the case that is currently open in the interface. Geochem-EZ does not allow the user to build cases with dissimilar components in the same interface.

Load file- opens a Geochem-EZ "Geo" file.

Open / Save DAT file- allows the user to handle "DAT" files created in Geochem-PC. Conversion of "DAT" files to the Geochem-EZ "Geo" file format is recommended.
