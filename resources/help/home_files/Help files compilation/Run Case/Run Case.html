Run Case-

After all the case information has been entered, the user should begin calculations by clicking on "Run Case". Geochem-EZ will open the "Output Viewer" and there will be output folders for each of the cases that have been run. Clicking on an output folder will show all of the output tables that have been generated. One can choose output tables in the Advanced Options section. The default output tables are Case Progress, Overall Speciation, and Primary Percentage Distribution. See Advanced Options and Output Tables for more information on the other output tables.
