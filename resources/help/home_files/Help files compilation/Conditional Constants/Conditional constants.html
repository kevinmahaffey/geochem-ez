Conditional Constants-

These are the final, ionic strength-adjusted that GEOCHEM uses in its calculations.  They are iteratively adjusted when Ionic strength (I) is being computed as part of the calculation, but obviously fixed at the outset when Ionic strength is fixed.  Their utility during routine use is pretty minimal, but they can be helpful when making adjustments to (or troubleshooting) the database.  For example, if you start with a constant at I=0.1 from the Smith and Martell compilation, convert it to I=0, you should get the original value back in this table during a run where I is set to 0.1. 

See: Output Tables for more information on the other output tables.
